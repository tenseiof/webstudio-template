# WebStudio Template

A simple responsive website template built with HTML, CSS, and JavaScript.

## Live Demo

https://tenseiof.github.io/webstudio-template/

## Responsive Design

This template includes basic responsive support for:

- Mobile: 480px
- Tablet: 768px
- Desktop: 1200px

## Technologies Used

- HTML5
- CSS3 (Sass included)
- JavaScript (vanilla)

## Features

- Simple layout
- Clean structure
- Responsive breakpoints
- Ready-to-use template
